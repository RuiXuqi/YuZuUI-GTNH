# YuZuUI-GTNH
使用千恋万花标题界面替换 Minecraft 标题界面.  --修改自[1.20.1 by IMG](https://github.com/ming-sc/YuZuUI-Forge)

minecraft 1.7.10 移植版

因原仓库许可证不明，所以本项目也不知道该用啥许可证
